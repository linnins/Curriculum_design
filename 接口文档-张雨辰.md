#### 1.查询可用座位
* 基本信息 
	> 请求路径；/seat/query
	> 请求方式：GET
	> 接口描述：查询现在的可用座位
	
* 请求参数
	| 参数名  | 类型   | 是否必须 | 备注 |
    | ----    | ----   | ----     | ---- |
    | token   | string | 是       |      |
    | place   | number | 是       | 分区 |
    | hour_f  | number | 是       | 开始小时 |    (0-11)
    | min_f   | number | 是       | 开始分钟 |    (0-59)
    | hour_t  | number | 是       | 结束小时 |    (0-11)
    | min_t   | number | 是       | 结束分钟 |    (0-59)

请求样例
GET http://xxx/seat/query/?token=xxx&place=1&hour_f=6&min_f=30&hour_t=7&min_t=10 

* 相应参数
	| 名称   | 类型    | 备注                            |
	| ----   | ----    | ----                            |
	| code   | number  | 响应码 0为无权限 1为成功 2为失败|
	| plc    | string  | 返回一个0/1串 0表示无人         |

* 响应数据样例
``` json
{
	"code" : 1,
	"plc"  : 1010111011,
}
```


#### 2.查询座位是否可用 
* 基本信息 
	> 请求路径；/seat/queryseat 
	> 请求方式：GET
	> 接口描述：查询座位是否可用 
	
* 请求参数
	| 参数名  | 类型   | 是否必须 | 备注 |
    | ----    | ----   | ----     | ---- |
    | token   | string | 是       |      |
    | place   | number | 是       | 分区 |
    | seat    | number | 是       | 座位 |
    | hour_f  | number | 是       | 开始小时 |    (0-11)
    | min_f   | number | 是       | 开始分钟 |    (0-59)
    | hour_t  | number | 是       | 结束小时 |    (0-11)
    | min_t   | number | 是       | 结束分钟 |    (0-59)

请求样例
GET http://xxx/seat/queryseat/?token=xxx&place=1&seat=1&hour_f=6&min_f=30&hour_t=7&min_t=10 

* 相应参数
	| 名称     | 类型    | 备注                                          |
	| ----     | ----    | ----                                          |
	| code     | number  | 响应码 0为无权限 1为成功 2为失败 3 为非法输入 |
	| seat     | number  | 0表示无人 1表示有人                           |
	| time_cnt | number  | time中的占用个数                              |
	| time     | string  | 有人的时间 两个字节拼成一个时间 8字节为一组   | 

* 响应数据样例
``` json
{
	"code"     : 1,
	"seat"     : 1,
	"time_cnt" : 2,
	"time"     : 1012121220102122,
}
```

#### 3.预约座位 
* 基本信息 
	> 请求路径；/seat/getseat 
	> 请求方式：GET
	> 接口描述：查询座位是否可用 
	
* 请求参数
	| 参数名  | 类型   | 是否必须 | 备注     |
    | ----    | ----   | ----     | ----     |
    | token   | string | 是       |          |
    | place   | number | 是       | 分区     |
    | seat    | number | 是       | 座位     |
    | hour_f  | number | 是       | 开始小时 |    (0-11)
    | min_f   | number | 是       | 开始分钟 |    (0-59)
    | hour_t  | number | 是       | 结束小时 |    (0-11)
    | min_t   | number | 是       | 结束分钟 |    (0-59)
    

请求样例
GET http://xxx/seat/getseat/?token=xxx&place=1&seat=1&hour_f=6&min_f=30&hour_t=7&min_t=10 

* 相应参数
	| 名称   | 类型    | 备注                                        |
	| ----   | ----    | ----                                        |
	| code   | number  | 响应码 0为无权限 1为成功 2为有人 3为非法输入|
	| msg    | string  | 返回提示信息                                |
 
* 响应数据样例
``` json
{
	"code" : 1,
	"msg"  : "成功预约",
}
```
